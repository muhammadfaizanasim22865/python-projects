## 🔍 Project Details

### 🤖 Jarvis AI Assistant
A text-based AI assistant inspired by Iron Man's JARVIS. Responds to user commands and performs basic tasks — an early exploration into intelligent systems and natural language interaction.

---

### 🏥 Emergency Room Simulation
Simulates an emergency room using a **priority queue** data structure. Patients are assigned priority levels and served accordingly — directly applying DSA concepts to a real-world scenario.

---

### 🎫 Ticket Counter Simulation
Models a real-world ticket counter using **queue (FIFO)** logic. Demonstrates how data structures like queues manage sequential real-world processes efficiently.

---

### 📚 Library Management System
A full-featured library system built with **Object-Oriented Programming**. Supports adding books, issuing, returning, and managing records — showcasing classes, objects, and encapsulation.

---

### 🏠 Smart Home Control System
A simulated smart home controller that manages virtual devices like lights, fans, and locks. Demonstrates OOP design patterns applied to an IoT-inspired scenario.

---

### 🌐 Internet Speed Tester
A utility tool that tests and displays your current internet download/upload speed — practical and straightforward use of Python libraries.

---

### 🔮 Python Fortune Teller
A fun program that uses Python's randomness to predict fortunes. Lightweight and creative — an exploration of randomness and probability in code.

---

### 👽 Alien Language Translator
Translates normal English text into a custom-designed alien language using character mapping and string manipulation. A creative application of Python string operations.

---

### 🤫 Secret Fizzy Language
A personal secret language encoder and decoder — named after my own nickname "Fizzy." Encodes messages into a custom cipher and decodes them back.

---

### 😂 Python Prank
A harmless, funny prank program written in Python. Built for fun while learning control flow, loops, and user interaction.
